# EShop

E-commerce web application built with HTML, CSS, JavaScript, jQuery, and Bootstrap.

## Features

- Product catalog with filtering by gender, category, brand, section, price, and color
- Product sorting (name, price, discount) and search
- Pagination (Load More / Show Less)
- Shopping cart and wishlist with localStorage persistence
- Product detail page with related products
- Blog section with comments
- Checkout form with validation
- Responsive design for all screen sizes

## Tech Stack

- HTML5, CSS3, JavaScript (ES6)
- jQuery 1.11.3
- Bootstrap 3
- Font Awesome 6
- JSON files as data source (no backend)

## Getting Started

1. Clone the repository
2. Open with VS Code
3. Run with Live Server extension
4. Open `http://localhost:5500` in your browser
